# SalesTracker

Настольное приложение для сбора и агрегации данных о заказах с маркетплейсов
**Wildberries** и **Ozon**. Подсчитывает количество проданных единиц каждого
артикула за выбранный период и показывает результат в виде наглядных отчётов.

Реализовано на **Java + Swing**, данные хранятся в локальной встроенной БД **H2**,
сборка — через **Apache Maven**.

---

## Возможности

- привязка магазинов Wildberries и Ozon по API-ключам;
- ручная и автоматическая (раз в час) синхронизация заказов;
- отчёт за день: «артикул — название — маркетплейс — количество»;
- сводный отчёт за период (от 1 до 90 дней);
- фильтрация отчётов по магазинам;
- аналитика: топ-N артикулов, сравнение двух периодов, артикулы с падением продаж;
- экспорт любого отчёта в **XLSX** и **CSV**.

---

## Требования

- **JDK 17 или выше** (проверено также на JDK 24);
- **Apache Maven 3.8+**;
- подключение к интернету — для обращения к API маркетплейсов.

Проверить, что всё установлено:

```bash
java -version
mvn -v
```

## Установка и запуск

### 1. Установить JDK 17+

- **Windows:** скачайте установщик с [adoptium.net](https://adoptium.net/) и установите.
- **Linux:** `sudo apt install openjdk-17-jdk` (или аналог для вашего дистрибутива).

### 2. Установить Apache Maven

- Скачайте с [maven.apache.org](https://maven.apache.org/download.cgi),
  распакуйте и добавьте папку `bin` в переменную окружения `PATH`.

### 3. Собрать проект

В корневой папке проекта (где лежит `pom.xml`) выполните:

```bash
mvn clean package
```

После сборки в папке `target/` появится файл **`salestracker.jar`**
(со всеми зависимостями внутри).

### 4. Запустить приложение

```bash
java -jar target/salestracker.jar
```

При первом запуске в папке `data/` автоматически создаётся файл базы данных
`salestracker.mv.db` — отдельно устанавливать СУБД не нужно.

## Первые шаги

1. Откройте вкладку **«Настройки»** и привяжите магазин:
   - для **Wildberries** введите название и API-ключ;
   - для **Ozon** введите название, **Client-Id** и **Api-Key**.
2. Нажмите кнопку **«Синхронизировать»** вверху окна — приложение загрузит
   заказы за последние 30 дней.
3. Перейдите на вкладку **«Отчёт за день»** или **«Отчёт за период»**
   и постройте отчёт.

Подробнее — в [руководстве пользователя](docs/user-guide.md).

## Где взять API-ключи

- **Wildberries:** личный кабинет продавца → «Настройки» → «Доступ к API» →
  создать токен с категорией «Статистика».
- **Ozon:** личный кабинет продавца → «Настройки» → «Seller API» →
  значения **Client-Id** и **Api-Key**.
